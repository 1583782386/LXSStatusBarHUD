# LXSStatusBarHUD
状态栏指示器
## 显示成功信息
```obj
 [LXSStatusBarHUD showSuccess:@"加载成功 ！"];
```
## 显示普通信息
```obj
 [LXSStatusBarHUD showMessage:@"普通信息 ！"];
```

## 显示失败信息

```obj
 [LXSStatusBarHUD showError:@"加载失败 ！"];
```
## 显示正在加载信息

```obj
 [LXSStatusBarHUD showLoading:@"加载中。。。"];
```
## 隐藏
```obj
 [LXSStatusBarHUD hide];
```
